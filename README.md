# Patio Almeyda — Dashboard de Negocio

Dashboard interactivo con el análisis de ventas de Patio Almeyda Bar (jul 2021 – jul 2026).

**Ver el sitio:** abrir `index.html` o, con GitHub Pages activado, en
`https://claudinhostgo-byte.github.io/PatioAlmeyda2026/`

## Secciones

- **Resumen** — KPIs globales del negocio
- **Ventas** — series mensuales, estacionalidad, comparativas por año
- **Productos** — categorías, top productos, márgenes
- **Operaciones** — días, horarios, zonas, duración de visitas
- **Equipo** — desempeño por mesero
- **Financiero & Caja** — P&L, food cost, propinas, productividad
- **Contexto & Mercado** — correlación con IMACEC, IPC, dólar y desempleo (fuentes: Banco Central de Chile / INE)
- **Recomendaciones** — insights accionables

## Notas técnicas

- Archivo único autocontenido (`index.html`); única dependencia externa: Chart.js vía CDN.
- Datos agregados embebidos en el HTML. Los datos crudos de transacciones y los scripts de extracción **no** se publican en este repositorio.
- Última actualización de datos: 6 de julio de 2026.
